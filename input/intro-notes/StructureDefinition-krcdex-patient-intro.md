### Patient Identifier

Patient의 identifier의 유형에 따라 아래 내용을 따름

- 환자 번호
  - type
    - system : http://terminology.hl7.org/CodeSystem/v2-0203
    - code : MR
  - system : 요양기관 OID
  - value : 환자번호
- 주민등록번호
  - type
    - system : http://terminology.hl7.org/CodeSystem/v2-0203
    - code : NNKOR
  - system : https://mois.go.kr/rrn
  - value : 주민등록번호
- 마이헬스웨이
  - type
    - system : http://terminology.hl7.org/CodeSystem/v2-0203
    - code : MHID
  - system : https://myhealthway.go.kr
  - value : 마이헬스웨이 ID
